# ResumeVita – Resume Builder

ResumeVita is a full-stack web application that allows users to create professional resumes quickly and easily.

## 🚀 Live Project

Frontend: https://resumevita.vercel.app/
Backend API: https://resume-backend-cduz.onrender.com

## ✨ Features

* User Signup & Login
* Resume Creation Dashboard
* Professional Resume Layout
* Data stored in MongoDB
* Secure authentication with JWT
* Fully deployed (Frontend + Backend)

## 🛠 Tech Stack

Frontend:

* React
* Vite
* Axios
* CSS

Backend:

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## 📦 Installation (Local Setup)

1. Clone the repository
2. Install dependencies

```
npm install
```

3. Run development server

```
npm run dev
```
